# PropReady-Khayalami

A comprehensive property platform for buyers, sellers, and real estate agents in South Africa.

## Features

- **Buyer Portal**: Learning modules, property search, bond calculator, and pre-qualification
- **Seller Portal**: Learning modules, property valuation booking, commission calculator, and seller dashboard
- **Agent Portal**: Lead management, property listings, viewing scheduling, and agent dashboard
- **Interactive Tools**: Commission calculator, transfer cost calculator, and bond calculator
- **Modern UI/UX**: Beautiful, responsive design with gradient styling and smooth animations

## Tech Stack

- Next.js 15
- React 18
- TypeScript
- Tailwind CSS
- Lucide React Icons

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

This project is configured for deployment on Netlify. See `netlify.toml` for configuration details.
